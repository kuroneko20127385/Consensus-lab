How to run:

từ scenario 1 -> 6:
python main.py --scenario <num> --seed <s> --rounds <r> 

--scenario: số kịch bản (1–6).
--seed: seed cho random (giúp tái lập kết quả).
--rounds: số vòng chạy.

Scenario 7:
python main.py --scenario 7 --base7 <num> --k <num> --seed<s> --rounds <r>

--base7: số kịch bản (1–6).
--seed: seed cho random (giúp tái lập kết quả).
--rounds: số vòng chạy.
--k: hệ số finality
